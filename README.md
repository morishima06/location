# 機能一覧
- 検索機能
  - キーワード検索
  - キーワード履歴検索
  - ブランド検索
  - ブランド一覧表示
  - エリア検索
  - エリア一覧表示
- Google Map
  - 店舗情報に合わせてのMapを表示
- ログイン機能
  - 店舗情報の登録、編集
  - ブランド、エリアの追加

# 主な使用技術・言語
- React18.2
  - eslint
  - prettier
- PHP8.2.15
- Laravel9.52.7
 - laravel breeze
- Inertia
- tailwind3.2.1
- Mysql
- Docker(Docker Compose)
- Vite

# 機能一覧
- 検索機能
  - キーワード検索
  - キーワード履歴表示
  - ブランド検索
  - ブランド一覧表示
  - エリア検索
  - エリア一覧表示
- Google Map
  - 店舗情報に合わせてのMapを表示
- ログイン機能
  - 店舗情報の登録、編集
  - ブランド、エリアの追加
